Introducción
============

El proyecto ``campo_estatico_mdf`` implementa un solucionador de la ecuación
de Laplace en dos dimensiones mediante métodos numéricos clásicos.

Características:

- Solución usando método iterativo de Jacobi
- Definición flexible de condiciones de frontera
- Cálculo del campo eléctrico
- Visualización del potencial y del campo
- Arquitectura modular para extensiones futuras
